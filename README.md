# InvasiveSpeciesCPP
C++ implementation for [InvasiveSpecies](https://github.com/yanfrimmel/InvasiveSpecies) 

## How to use

### To build on Linux

```bash
    mkdir build
    cd build
    cmake ..
    make
```

### To run

```bash
    cd build
    ./InvasiveSpeciesCPP {SREEN_WIDTH} {SCREEN_HEIGHT} {IS_FULL_SCREEN}
